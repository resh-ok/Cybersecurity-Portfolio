# Linux Fundamentals for Cybersecurity

## Overview

This repository documents my hands-on Linux learning journey through a series of practical labs focused on the foundational skills required in cybersecurity. Each lab includes objectives, commands used, observations, security insights, and screenshots to demonstrate practical experience.

The labs are based on Linux concepts commonly covered in entry-level cybersecurity training, including the Google Cybersecurity Certificate, and emphasize real-world command-line usage.

## Learning Objectives

Through this repository, I aim to:

* Develop strong Linux command-line skills.
* Understand Linux file system navigation and management.
* Learn file permissions and access control.
* Search and analyze files and log data.
* Manage processes and monitor system activity.
* Explore basic networking commands.
* Practice log analysis and file archiving.
* Build a cybersecurity-focused Linux portfolio.

## Key Takeaways

Completing these labs strengthened my understanding of Linux fundamentals and how they apply to cybersecurity. I gained practical experience navigating the file system, managing files and permissions, analyzing logs, monitoring processes, inspecting network configurations, and working with archives. These skills provide a solid foundation for more advanced topics such as security monitoring, incident response, digital forensics, and automation.
