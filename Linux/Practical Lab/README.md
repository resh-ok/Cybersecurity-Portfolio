# Linux Fundamentals for Cybersecurity Labs

## Overview

This repository contains a collection of hands-on Linux labs completed as part of my cybersecurity learning journey. The labs focus on fundamental Linux skills that are essential for system administration, security operations, incident response, and digital forensics.

The exercises are inspired by the Linux concepts covered in the Google Cybersecurity Certificate and emphasize practical command-line experience through step-by-step activities.

## Objectives

The goals of this repository are to:

* Build a strong foundation in Linux.
* Develop practical command-line skills.
* Learn Linux file system management.
* Understand file permissions and access control.
* Practice searching and analyzing files and logs.
* Explore Linux networking and process management.
* Create a portfolio demonstrating hands-on Linux experience.

## Labs Included

| Lab | Topic                               | Skills Covered                                 |
| --- | ----------------------------------- | ---------------------------------------------- |
| 1   | Linux Terminal Basics               | `echo`, `whoami`, `id`, terminal usage         |
| 2   | File System Navigation              | `pwd`, `ls`, `cd`                              |
| 3   | Creating Files and Directories      | `mkdir`, `touch`, `cp`, directory organization |
| 4   | Viewing and Reading Files           | `cat`, `less`, `head`, `tail`                  |
| 5   | Copying, Moving, and Deleting Files | `cp`, `mv`, `rm`, `rmdir`                      |
| 6   | Searching for Files                 | `find`                                         |
| 7   | Searching File Contents             | `grep`                                         |
| 8   | Linux File Permissions              | `ls -l`, `chmod`                               |
| 9   | Managing Processes                  | `ps`, `top`, `jobs`, `kill`                    |
| 10  | Basic Networking Commands           | `ip`, `ping`, `ss`                             |
| 11  | Viewing System Logs                 | `journalctl`, log analysis                     |
| 12  | Archiving and Compressing Files     | `tar`, `gzip`                                  |
